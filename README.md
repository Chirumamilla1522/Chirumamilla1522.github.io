### 💻 About This Repository (My Portfolio)

This repository also contains the source code for my [personal portfolio website](https://Chirumamilla1522.github.io). It is designed to be a lightweight, highly performant static site.

**Features:**
- **Neo-Brutalist Design:** Clean, oversized typography using Space Grotesk.
- **Native Dark Mode:** Seamless light/dark theme toggling with local storage memory.
- **Zero Dependencies:** Built entirely with vanilla HTML, CSS, and JavaScript for instant load times.
