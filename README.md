# Bhargav Chirumamilla - Personal Portfolio

Live site: [https://Chirumamilla1522.github.io](https://Chirumamilla1522.github.io) *(Update this link once your site is live!)*

This repository contains the source code for my personal portfolio website. It is designed to be a lightweight, highly performant static site that highlights my experience in Applied AI, scalable Machine Learning infrastructure, and production deployments.

## ⚡ Features

- **Neo-Brutalist / Editorial Design:** Clean, oversized typography using Space Grotesk.
- **Native Dark Mode:** Seamless light/dark theme toggling with local storage memory.
- **Scroll Reveal Animations:** Smooth, intersection-based fade-in effects as the user scrolls.
- **Zero Dependencies:** Built entirely with vanilla HTML, CSS, and JavaScript. No bulky frameworks, no build steps, instant load times.
- **Fully Responsive:** Optimized for desktop, tablet, and mobile viewing.

## 🛠 Tech Stack

- **HTML5** (Semantic structure)
- **CSS3** (Custom properties, CSS Grid, Flexbox, Media Queries)
- **Vanilla JavaScript** (DOM manipulation, local storage, scroll event listeners)
